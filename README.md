# Аналіз даних Metrocar

“Metrocar” — це сервіс для виклику автомобілів, подібний до Uber чи Bolt. Він дозволяє користувачам завантажити мобільний додаток, пройти реєстрацію, замовити машину, здійснити поїздку, оплатити її та залишити свій відгук.

Мета — провести аналіз користувацької воронки, щоб визначити, на яких етапах і з якої причини відбувається втрата користувачів.

Використані технології: SQL, Tableau.

Аналіз даних охоплює такі аспекти:

● завантаження додатка з урахуванням платформи та вікової категорії користувачів;

● реєстрація нових користувачів із прив’язкою до сесії завантаження;

● усі події, що стосуються поїздок, від моменту запиту до завершення або скасування;

● транзакції, включно з сумами платежів та статусами виконання;

● відгуки користувачів, які містять тексти й оцінки.

Створено 2 [дашборди](https://public.tableau.com/app/profile/a.a8255/viz/Metrocarpublic/Metrocar1) Tableau (понад 7 діаграм) для комплексного аналізу воронки конверсії та операційної ефективності сервісу таксі. Візуалізація дозволила сегментувати аудиторію за платформами й демографією та реалізувати моніторинг клієнтського досвіду (CSAT) та погодинної активності.
